# **App Name**: OrderFlow

## Core Features:

- Order Management: Create, read, update, and delete orders in the system.
- Real-time Totals Calculation: Automatically calculate the order total as products are added or modified.
- WhatsApp Integration: Generate WhatsApp links for customers, automatically sanitizing phone numbers.
- Conditional Address Update: Automatically update the shipping address based on the selected delivery service.
- Date Synchronization: Automatically set the 'entregaLimite' to the same value as 'entrega' by default.
- Order Dashboard with Search: Display orders in a searchable table with color-coded status badges.
- Material Availability Tracking: Track the availability of materials for each product using checkboxes.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) for a professional and trustworthy feel.
- Background color: Light gray (#E9E9F2) for a clean, minimalist look.
- Accent color: Deep purple (#5E2A84) for highlighting key actions and elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral aesthetic. Note: currently only Google Fonts are supported.
- Use simple, consistent icons from a set like Feather or Material Icons.
- Employ a clean, minimalist layout optimized for mobile responsiveness in a workshop environment, using Tailwind CSS grid and flexbox utilities.
- Incorporate subtle transitions for UI elements, providing user feedback
